# Struktur Folder Mandala

Struktur dasar direktori Mandala di setiap simpul:

mandala/

   ├── mandala.info

   ├── mandala.share

   ├── mandala.verse/

   ├── mandala.doc/

   ├── mandala.comm/

   ├── mandala.log/


Setiap folder memiliki fungsi tersendiri (dokumentasi, komunikasi, arsip bersama, dll.).
